Memosly 是一款功能丰富的 Memos 安卓客户端。Memos 是开源、自托管的备忘录中心，让你随时随地记录想法、灵感和笔记。

<b>核心功能</b>

- <b>富文本 Markdown 编辑器</b> — 完整支持 Markdown 语法，包括 GFM 表格、任务列表、代码块和行内格式。实时预览让你即时查看最终效果。

- <b>媒体附件</b> — 直接在备忘录中上传图片、视频和文件。图片内联显示，视频即时播放，附件一键下载。

- <b>标签系统</b> — 使用 # 标签整理备忘录。智能补全在输入时自动推荐已有标签，支持按标签快速筛选。

- <b>表情回应与评论</b> — 用 emoji 表情回应备忘录，添加评论进行讨论。非常适合共享的 Memos 实例。

- <b>搜索与发现</b> — 全文搜索所有备忘录。在发现页浏览公开的备忘录。

- <b>归档管理</b> — 归档暂时不需要的备忘录，随时恢复或删除。

- <b>深色模式</b> — 精美的浅色和深色主题，跟随系统设置自动切换。

- <b>多语言支持</b> — 支持中文和英文，自动检测系统语言。

- <b>灵活导航</b> — 可选择侧边栏抽屉或底部标签页导航，满足不同使用习惯。

- <b>多版本兼容</b> — 支持 Memos 服务器 v0.24、v0.25 和 v0.26+。已在 v0.24.4、v0.25.3 和 v0.26.1 上完整测试，其他小版本应可正常使用但未经完整测试。

<b>关于 Memos</b>

Memos (https://usememos.com) 是一个开源、自托管的备忘录中心，专注于隐私和简洁。部署自己的实例，完全掌控你的数据。

Memosly 是独立的第三方客户端，与 Memos 官方项目无关。
